# Auto Risk Scoring

End-to-end risk scoring framework for automobile insurance. This project is part of my Master's (M1) thesis and focuses on claim frequency modeling using Poisson regression, Negative Binomial models, and gradient boosting methods, with rigorous validation and model comparison.

⚠️ Due to academic requirements, the final version of the code will be released on May 4, 2026.
