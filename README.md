# Auto Risk Scoring

End-to-end risk scoring framework for automobile insurance, developed as part of my Master's (M1) thesis.

The project focuses on claim frequency modeling using a real-world portfolio of approximately 680,000 insurance contracts. It includes the implementation and comparison of statistical and machine learning models, notably Zero-Inflated Poisson and Random Forest, for risk prediction.

Model performance is assessed through rigorous cross-validation, with a focus on interpretability and identification of key drivers to support pricing and policyholder segmentation.
